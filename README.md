# Deque — упрощённая реализация std::deque

Реализация упрощённого аналога `std::deque<int>` **без использования стандартных контейнеров и умных указателей**.

Контейнер реализован на основе кольцевого буфера блоков фиксированного размера.

---

## Сложность операций:

| Операция             | Сложность |
| -------------------- | --------- |
| Индексация `[ ]`     | O(1)      |
| PushBack / PopBack   | O(1)      |
| PushFront / PopFront | O(1)      |
| Clear                | O(n)      |

---

## Основные идеи реализации

- Дек хранит данные в массиве блоков (чанков)
- Размер одного блока — **128 элементов (512 байт)**
- Используется кольцевая адресация
- Все ссылки на элементы остаются валидными при вставках и удалениях
  (кроме удаляемых элементов), как в `std::deque`

---

## Возможности

- Хранение элементов типа `int`
- Конструкторы:
  - по умолчанию
  - с заданным размером (заполнение нулями)
  - из `std::initializer_list`
  - копирования и перемещения
- Операторы присваивания (copy / move)
- Индексация за **O(1)**
- Вставка и удаление:
  - `PushBack` / `PopBack`
  - `PushFront` / `PopFront`
- Методы:
  - `Size`
  - `Capacity`
  - `Clear`
  - `Swap`

---

## Пример использования

```cpp
Deque d = {1, 2, 3};

d.PushFront(0);
d.PushBack(4);

for (size_t i = 0; i < d.Size(); ++i) {
    std::cout << d[i] << " ";
}
